# windows-server-nerdnest-lab-1
Windows Server infrastructure lab with pfSense, Active Directory, DNS and PowerShell user automation.

This project simulates a small company IT infrastructure using Windows Server, pfSense, Active Directory and DNS.

The lab demonstrates server deployment, domain configuration, network management and automated user provisioning with PowerShell.


## Lab Environment

Virtual machines created in VMware:

- pfSense Router
- Windows Server (Domain Controller)
- Windows 11 Client

Important remark: make snapshots to easily start over or correct mistakes. 

## Network Architecture

- Virtual network with pfSense router
- Windows Server static IP
- Client using DHCP

Domain name: nerdnest.test
